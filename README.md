# Classificador-de-imagens-de-feij-o
Neste pequeno projeto, apresento uma interface web feita em Django que detecta a presença de doenças em folhas de feijão, como ferrugem ou manchas angulares. Com base nisso, é gerado um prompt para a Mistral-7B-OpenOrca, que produz uma descrição para a doença encontrada.
